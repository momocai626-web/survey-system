=====================================
課程前準備事項 - 學員 Todo List
=====================================

請在上課前完成以下準備，讓我們能更順利地開始課程！

-------------------------------------
【1. 確認設備】
-------------------------------------
□ 自備筆電（macOS 或 Windows 皆可）
□ 確認筆電電力充足，建議攜帶充電器

-------------------------------------
【2. 帳號申請】
-------------------------------------
□ 申請 Google 帳號
  → https://accounts.google.com/signup

□ 申請 GitHub 帳號
  → https://github.com/signup
  → 記下你的帳號名稱，課堂會用到

-------------------------------------
【3. 軟體安裝】
-------------------------------------
□ 安裝 VS Code（Visual Studio Code）
  → https://code.visualstudio.com/download
  → 選擇對應你作業系統的版本下載安裝

□ 安裝 Git
  → Windows：https://git-scm.com/download/win
  → macOS：打開終端機輸入 git --version，若尚未安裝會自動提示

-------------------------------------
【4. 課程系統】
-------------------------------------
□ 開啟課程首頁，確認可以正常瀏覽：
  → https://momocai626-web.github.io/survey-system/

□ 填寫課前問卷（課程首頁 → 學員入口 → 課前問卷）

-------------------------------------
【準備完成確認】
-------------------------------------
上課前請確認：
  ✓ 筆電可以開機上網
  ✓ GitHub 帳號已申請完成
  ✓ VS Code 已安裝可以開啟
  ✓ 課前問卷已填寫

有任何問題請提前告知講師，謝謝！
