=====================================
課程大綱
AI 輔助網頁開發入門：從 0 到部署
=====================================

時長：2 小時
對象：完全無程式基礎的學員
目標：課程結束後能獨立將網頁部署到網路上

-------------------------------------
【課前】學員自行完成（上課前）
-------------------------------------
□ 申請 GitHub 帳號
□ 安裝 VS Code
□ 安裝 Git
□ 填寫課前問卷
  → https://momocai626-web.github.io/survey-system/

-------------------------------------
【第一段】破冰 + 工具介紹（20 分鐘）
-------------------------------------
■ 0:00 開場與自我介紹（5 分鐘）
  - 課程目標說明
  - 今天要做出什麼成果

■ 0:05 認識今天用的工具（15 分鐘）
  - VS Code 是什麼？為什麼用它？
  - GitHub 是什麼？雲端版的「存檔空間」
  - AI 助手 Qwen Code 是什麼？怎麼問它寫程式？
  - GitHub Pages 是什麼？免費架網站的方法

-------------------------------------
【第二段】動手實作 - 用 AI 寫網頁（50 分鐘）
-------------------------------------
■ 0:25 認識 VS Code 介面（10 分鐘）
  - 開啟資料夾、新增檔案
  - 安裝 Qwen Code 擴充元件

■ 0:35 用 AI 生成第一個網頁（20 分鐘）
  - 打開 Qwen Code，輸入需求
  - 用詠唱（Prompt）請 AI 寫 HTML
  - 在瀏覽器預覽結果
  - 修改內容、看到畫面即時變化

■ 0:55 將網頁存到 GitHub（20 分鐘）
  - 建立 GitHub Repository
  - 用 VS Code 上傳檔案（Git push）
  - 確認檔案出現在 GitHub 上

-------------------------------------
【第三段】部署上線（30 分鐘）
-------------------------------------
■ 1:25 啟用 GitHub Pages（15 分鐘）
  - Repository → Settings → Pages
  - 選擇分支，儲存設定
  - 等待部署，取得網址

■ 1:40 成果展示與調整（15 分鐘）
  - 用手機掃 QR Code 打開自己的網站
  - 修改內容 → push → 網站自動更新
  - 學員互相分享網址

-------------------------------------
【第四段】收尾（20 分鐘）
-------------------------------------
■ 1:55 生命徵象量測實作（10 分鐘）
  - 開啟課堂練習頁面
  - 輸入量測數值，觀察異常提示與 BMI 換算

■ 2:05 Q&A + 填寫課後問卷（10 分鐘）
  - 開放提問
  - 填寫課後問卷
    → https://momocai626-web.github.io/survey-system/

-------------------------------------
【課後】延伸學習方向
-------------------------------------
- 修改網頁樣式（CSS）
- 加入更多功能（表單、互動效果）
- 串接 Google Sheets 收集資料
- 探索更多 AI 輔助開發工具

=====================================
講師備註
=====================================
- 每段結束預留 2–3 分鐘確認學員進度
- Qwen Code 如回應過慢，可切換示範模式（講師螢幕投影）
- Git push 步驟最容易卡關，建議提前準備錯誤排解 SOP
- 課前問卷結果可即時投影，作為破冰話題
